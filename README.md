# Sales Data Analysis using Python

## Tools Used
- Python
- Pandas
- Matplotlib

## Objective
Analyze sales data to identify total sales, category-wise sales, and top-performing products using Python.

## Key Insights
- Electronics category generated the highest sales
- Laptop and Mobile are the top-selling products
- Total sales calculated using derived Total_Sales column

## Files in This Project
- sales_data.csv – Dataset
- sales_analysis.py – Python analysis script
- sales_by_category_python.png – Visualization output

## Conclusion
This project demonstrates data analysis and visualization using Python libraries, suitable for entry-level data analyst roles.
